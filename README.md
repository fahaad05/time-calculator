# Time Calculator

A simple web tool to perform time-based calculations.

## Features

### 1. Add / Subtract Time
- Set a base time (e.g. 10:30)
- Add or subtract hours and minutes
- Handles 24-hour wrap (e.g. going past midnight)

Example:
10:30 + 2h 15m = 12:45

---

### 2. Time Difference
- Calculate the difference between two times
- Supports overnight calculations

Example:
23:00 → 02:00 = 3h 0m

---

### 3. Duration Calculator
- Add or subtract two durations
- Inputs are hours and minutes

Example:
2h 30m + 1h 45m = 4h 15m

---

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS

---

## Running locally

```bash
npm install
npm run dev
```
